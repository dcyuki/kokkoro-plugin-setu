> 好色哦，我都不看这些的

## 安装

``` shell
# 切换至 bot 目录
cd bot

# 安装 npm 包
npm i kokkoro-plugin-setu
```

在 [kokkoro](https://github.com/dcyuki/kokkoro) 成功运行并登录后，发送 `>plug on setu` 即可启用插件
